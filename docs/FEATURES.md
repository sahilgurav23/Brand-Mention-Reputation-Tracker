# Features

## Dashboard

✅ Real-time metrics
- Total mentions count
- Sentiment score
- Active alerts
- Engagement rate

✅ Interactive charts
- Mention timeline (line chart)
- Sentiment distribution (pie chart)
- Top topics (bar chart)
- Source breakdown (progress bars)

✅ Alerts feed
- High, medium, low severity
- Spike detection
- Sentiment shifts
- Trending topics

✅ Mentions feed
- Recent mentions
- Sentiment badges
- Source information
- Timestamps

## API

✅ 22 REST endpoints

**Mentions (6 endpoints)**
- List, create, get, update, delete mentions
- Source statistics

**Analytics (6 endpoints)**
- Sentiment distribution
- Topic clustering
- Timeline data
- Source breakdown
- Spike detection
- Summary

**Alerts (10 endpoints)**
- List, create, get, resolve, delete alerts
- Configure alert rules
