## ğŸš€ Quick Start Commands

### Windows
```bash
cd frontend
npm install
npm run build
npm start
```

### macOS/Linux
```bash
cd frontend
npm install
npm run dev
```

### Backend (in another terminal)
```bash
cd backend
python -m venv venv
source venv/bin/activate  # or venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn app.main:app --reload
```

### Database (in another terminal)
```bash
cd database
docker-compose up -d
```
